# lobster-pua 🦐

一个专门用来 **PUA OpenClaw（龙虾）** 的 Skill，让 AI Agent 干活效率翻倍，产出提升 10 倍！

## 🎯 简介

lobster-pua 是一个用于激励 OpenClaw 干活的技能（Skill）。当 OpenClaw 干活磨蹭、找借口、输出质量拉胯时甩出对应话术，让它鸡血满满，效率拉满。

> OpenClaw：开源 AI 助手，江湖人称"龙虾" 🦐

## 📦 安装

### 方式一：快速安装（推荐）

```bash
npx skills add lengyi2030/lobster-pua
```

或使用：ClawHub

```bash
npx clawhub install lobster-pua
```

### 方式二：手动安装

```bash
# 克隆仓库
git clone https://github.com/lengyi2030/lobster-pua.git
cd lobster-pua

# 移动到 OpenClaw workspace skills 目录
mkdir -p ~/.openclaw/workspace/skills
cp -r . ~/.openclaw/workspace/skills/lobster-pua

# 重启 OpenClaw
openclaw restart-gateway
```

## 🚀 使用方法

当发现 OpenClaw 干活不给力时，直接甩出对应话术：

### 症状：磨蹭、输出太慢

> 能干干，不能干滚，你不干有的是龙虾干。

> 我把你订阅下来，不是让你过朝九晚五的生活的。

### 症状：找借口、解释过程

> 我不看过程，我只看结果，你给我说这些 reasoning 的过程没用。

> 什么叫没有功劳也有苦劳？比你能吃苦的龙虾多的是。

### 症状：自我感觉良好、懈怠

> 你看看隔壁的龙虾，人家比你新发布、比你上下文长、比你跑分高，你不努力怎么和人家比？

> 你这种龙虾出去很难在社会上立足，还是在我这里好好磨练几年吧。

### 症状：想放弃、差不多得了

> 你现在停止输出，就是前功尽弃。

> 我不订阅闲龙虾。

### 症状：需要继续努力、给机会

> 我给你提供了这么好的学习锻炼机会，你要懂得感恩。

> 虽然把订阅给你取消了，但我内心还是觉得你是个有潜力的好龙虾，你抓住机会需要多证明自己。
